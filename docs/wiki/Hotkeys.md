# Hotkeys

Bindable in **Options → Controls → Key Binds → Loominary**. All unbound by default so nothing collides with your setup.

| Action | Equivalent |
|---|---|
| Steal map at crosshair | `/loominary import steal` |
| Preview active tile on crosshair map | `/loominary preview` |
| Revert previewed map at crosshair | `/loominary revert` |
| Switch to next incomplete tile | `/loominary tile next` |
| Show batch status | `/loominary status` |
| Toggle Loominary decoding (raw/decoded) | — (view the map with/without decoding) |
| Balance carpets to Litematica materials | `/loominary carpets balance` |
| Fill carpet stacks from nearby chests | `/loominary carpets fill` |
| Toggle duty-cycle auto-walk | `/loominary walk` |
| Stop all Loominary automation | `/loominary stop` |

The **decode toggle** is the one with no command equivalent: it flips between the decoded art and the raw carpet-colored map — handy for checking what players without the mod see.
